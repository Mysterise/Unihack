# Unihack
unihack syd 2018

## server setup
```
$ npm i -g yarn
$ cd server
$ yarn
```

**start dev** (auto-reloading)
```
$ yarn dev
```

**build** (compile ts)
```
$ yarn build
```

**lint**
```
$ yarn test:lint
```

## client setup
```
$ npm i -g yarn
$ cd client
$ yarn
```

**dev**
```
$ yarn dev
$ open 'http://localhost:8080'
```

**build**
```
$ yarn build
```
